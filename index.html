Welcome to the documentation of OpenSTEF!¶

OpenSTEF is a Python package which is used to make short term forecasts for the energy sector. It contains all components for the machine learning pipeline required to make a forecast. In order to use the package you need to provide your own data storage and retrieval interface. OpenSTEF is available at: https://pypi.org/project/openstef/.

Get started using the Quickstart or by reading about the Concepts.
